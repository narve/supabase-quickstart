Supabase Quickstart

What is this?

This is a simple way of quickly getting started with Supabase. It does not require any setup or installation - just get your Supabase configuration from the Supabase administration panel, input them here, and test your Supabase instance :)

If you are really lazy, you can play around with the default Supabase configuration, which is my own Supabase playground application.

This code is deliberately kept simple: No frameworks, no fanciness, limited functionality.
